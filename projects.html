My Work

Surrey On The Go

An application built using Ruby on Rails and Materialize for internal use for the non-profit agency, Surrey Services for Seniors, to coordinate their transportation services. The application matches seniors requesting rides to volunteer drivers based on availability, location, and other considerations.

Github | App


Flitter

A twitter-style microblog built using Sinatra and PostgreSQL. Users can create 'fleets' and follow other users. When logged in, a user sees a customized feed on the home page based on the users they are following.

Github | App


Bloggin'

A basic blog application built on Ruby on Rails as an exercise in models and associations. Users can create, edit, and delete posts and comments on others' posts. I set up a self referential model to allow for unlimited nested comments, similar to Reddit.


Github | App
